Quick Start
  1. Clone the repo and install dependencies (pip install -r requirements.txt).
  2. Set up your .env file (see README.md).
  3. Start MongoDB and Ollama locally.
  4. Run: uvicorn app.main:app --reload
  5. Open this page or the API docs.
